Creating Module

You can create a new module with the New Module Wizard(File->New->GWT Module)

You have to enter the package where you want to create the module and the module name. Cypal Studio for GWT will create the necesary artifacts for the module.

If you have given com.example.gwt.sample as the package and MyModule as the module name, the following artifacts will be created:

Packages
      com.example.gwt.sample
      com.example.gwt.sample.client
      com.example.gwt.sample.server

Java File
      'MyModule.java' in package 'com.example.gwt.sample'

Folder
      'public' in package 'com.example.gwt.sample'

HTML File
      'MyModule.html' in folder 'public'

Module File
      'MyModule.gwt.xml' in package 'com.example.gwt.sample'
