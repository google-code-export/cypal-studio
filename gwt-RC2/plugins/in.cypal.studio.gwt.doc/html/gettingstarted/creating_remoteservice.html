Creating Remote Service

Creating of a Remote Service involves creation of the following artifacts: If you have given the name as RemoteService and the uri as remoteService

Java Files
      'RemoteService.java' in package 'com.example.sample.client'
      'RemoteServiceAsync.java' in package 'com.example.sample.client'
      'RemoteServiceImpl.java' in package 'com.example.sample.server'

Cypal Studio for GWT, also adds an entry in your MyModule.gwt.xml and web.xml as well
