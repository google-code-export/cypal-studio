Creating Module

You can create a new module with the New Module Wizard(File->New->GWT Module)

You have to enter the package where you want to create the module and the module name. Cypal Studio for GWT will create the necesary artifacts for the module.

If you have given in.cypal.gwt.first as the package and MyFirstModule as the module name, the following artifacts will be created:

Packages
      in.cypal.gwt.first
      in.cypal.gwt.first.client
      in.cypal.gwt.first.server

Java File
      'MyFirstModule.java' in package 'in.cypal.gwt.first.client'

Folder
      'public' in package 'in.cypal.gwt.first.client'

HTML File
      'MyFirstModule.html' in folder 'public'

Module File
      'MyFirstModule.gwt.xml' in package 'in.cypal.gwt.first'
