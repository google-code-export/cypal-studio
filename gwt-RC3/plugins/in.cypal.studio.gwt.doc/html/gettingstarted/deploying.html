Deploying on an External Server

If you don't want to use the embedded Tomcat server and want to utilize some other web server, you can do it easily with Cypal Studio for GWT.

  • First configure the server in the Servers view.
  • Right click the configured server and select 'Add/Remove Projects...'
  • Add your project from the 'Available Projects' to 'Configured Projects'
  • Start the server. Your application will be published to the server
