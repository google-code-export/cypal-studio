Installing Cypal Studio for GWT

Cypal Studio for GWT needs Eclipse WTP. If you are seeing this page, you are already have it installed properly. One last thing you need to do is to set tell Cypal Studio for GWT where to find the gwt-*.jar files.(They are not shipped with Cypal Studio for GWT You have to download it from here)

Cypal Studio for GWT finds those jar files from the install location of GWT. Go to Window->Preferences->Cypal Studio to set the location.
