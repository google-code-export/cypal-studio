Running in Hosted Mode

You have to create a launch configuration to run a module in the hosted mode. Go to the Run dialog box and create a new launch configuration in the Gwt Hosted Mode Application category.

You can customize the launch configuration (such as module url, server port, JavaScript style) in the parameters page of the dialog. Once you are done, click on Run. This should launch two windows - one for the GWT Shell and the other for the browser.

The easiest way to run a module in hosted mode is to Right Click the module file and select 'Run As-> GWT Hosted mode Application'. If you already have a launch configuration for that module this will pick up the configuration and run, if not it will create one with the default values and save it for your future use and then launch the module.
