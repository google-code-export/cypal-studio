Creating Remote Service

Creating of a Remote Service involves creation of the following artifacts: If you have given the name as RemoteService and the uri as remoteService

Java Files
      'MyFirstRPC.java' in package 'in.cypal.gwt.first.client'
      'MyFirstRPC.java' in package 'in.cypal.gwt.first.client'
      'MyFirstRPC.java' in package 'in.cypal.gwt.first.server'

Cypal Studio for GWT, also adds an entry in your MyFirstModule.gwt.xml and web.xml as well
